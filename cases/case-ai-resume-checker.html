AI Resume Analyzer & Job Match Assistant

Repository: github.com/SaoodQureshi101/AI-Resume-Checker

Goal: Help job seekers quickly measure resume-job fit, identify missing skills, and improve resume quality.

Abstract

This project analyzes a resume against a target job description, finds alignment and gaps, and provides practical suggestions so applicants can improve role relevance before submitting.

Problem

Applicants often apply without knowing whether their resume content actually matches the role. This mismatch can reduce callbacks even when candidates have relevant experience.

Solution

  • Extract key skills and qualifications from the resume
  • Compare them to job-description requirements
  • Compute a semantic match score
  • Highlight missing or weakly represented skills
  • Suggest practical resume improvements

Methods

The app uses NLP and transformer-based semantic matching. Sentence-BERT embeddings are generated for resumes and job descriptions, then cosine similarity is used to evaluate alignment. A hybrid layer of deterministic skill extraction improves transparency and guidance.

AI Technology Used

  • sentence-transformers model: all-MiniLM-L6-v2
  • Hybrid approach: semantic embeddings + keyword skill extraction
  • Cosine similarity via scikit-learn
  • Optional guidance utilities in utils/ai_features.py

Tools and Technologies

  • Python
  • Hugging Face Transformers / Sentence-BERT
  • Streamlit
  • scikit-learn
  • PyPDF2
  • reportlab
  • requests (Data USA API salary lookup)

How to Run

  1. Open folder: cd "C:\Users\1\OneDrive\Desktop\ai-resume-analyzer"
  2. Create venv: python -m venv .venv
  3. Activate venv: .\.venv\Scripts\Activate.ps1
  4. Install deps: pip install -r requirements.txt
  5. Run app: streamlit run app.py

Note: Do not run with python app.py because this is a Streamlit app.
