## Quick publish to GitHub Pages

Repository: `https://github.com/SaoodQureshi101/Portfolio`

1. In this folder, run:

```bash
git init
git add .
git commit -m "Initial live portfolio"
git branch -M main
git remote add origin https://github.com/SaoodQureshi101/Portfolio.git
git push -u origin main
```

2. In GitHub, open repository settings:
   - Go to `Settings` -> `Pages`
   - Under `Build and deployment`, set `Source` to `Deploy from a branch`
   - Set branch to `main` and folder to `/ (root)`
   - Click `Save`

3. Wait for deployment (typically 1-3 minutes), then verify the green Pages status.

4. Live URL:

`https://saoodqureshi101.github.io/Portfolio/`

5. Verify these pages load from the live URL:
   - `/`
   - `/work.html`
   - `/about.html`
   - `/contact.html`
   - `/cases/case-campus-wayfinding.html`
   - `/cases/case-booking-flow.html`
   - `/cases/case-content-system.html`
   - `/cases/case-ai-resume-checker.html`
